## About Me

I build and study AI systems at the intersection of **financial decision-making**, **fairness**, **multi-agent LLM behavior**, and **lightweight deployable tools**.

My recent work leans toward a practical question:

> How can we make intelligent systems useful in real financial settings while keeping them robust, auditable, and fair?

I like projects that combine research ideas with runnable demos: clear problem framing, transparent evaluation, and code that helps other people reproduce the core workflow.

## Current Interests

- Financial LLM evaluation and decision-support systems
- Fairness, demographic robustness, and bias mitigation
- Multi-agent LLM behavior, especially herding and social influence
- Lightweight model deployment and practical AI tooling
- Desktop utilities built with Python, Electron, and web technologies

## Featured Projects

### [FinFair](https://github.com/ChangYM963/Finfair)

A public demo for **FinFair: A Regulatory-Aligned Robust Optimization Framework for Fair and Lightweight Financial Decision Models**.

Focus areas:

- demographic robustness in financial decision models
- counterfactual invariance under non-economic demographic perturbations
- adversarial learning and teacher-guided rational alignment
- lightweight financial AI under real deployment constraints

### [MAI Herding Test](https://github.com/ChangYM963/mai-herding-test)

A compact public demo for evaluating **herding behavior in financial multi-agent LLM systems**.

Focus areas:

- context-adaptive financial benchmark construction
- two-stage multi-agent evaluation
- Majority Alignment Index measurement
- mitigation analysis for social influence in agent systems

### [Transformers Code](https://github.com/ChangYM963/transformers-code)

A hands-on Transformers learning and practice repository covering NLP workflows, fine-tuning, PEFT, low-precision training, and distributed training.

Tech stack:

- PyTorch
- Transformers
- PEFT
- Datasets
- Accelerate
- bitsandbytes

### [Gemini Web Shell](https://github.com/ChangYM963/gemini-web-shell)

An unofficial Windows desktop wrapper for Gemini Web, built with Electron.

Focus areas:

- desktop app packaging
- web wrapper UX
- Windows development workflow
- practical AI assistant tooling

### [GitHub Followers Search](https://github.com/ChangYM963/Github_followers)

A small local desktop utility for loading and searching GitHub followers with real-time keyword and fuzzy matching.

Focus areas:

- GitHub REST API usage
- local caching
- fuzzy search
- simple productivity tooling
